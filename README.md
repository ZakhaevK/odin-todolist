# Important Objects:

## Todo-item Object Properties
Individual todo's will have several properties as per below:
- title
- description
- dueDate
- priority
- notes
- checklist

## Projects
Projects will essentially be lists of todo items.
Will likely be objects themselves with details such as:
- title
- description
- todoList
